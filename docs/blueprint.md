# **App Name**: Makuta Wallet

## Core Features:

- Multilingual Support: Allow users to switch between French, English, Spanish, Portuguese, Chinese, Lingala, Tshiluba, and Swahili.
- Dashboard: Display a consolidated view of multi-currency balances, transaction history, and quick access to key functions like scanning QR codes, sending/receiving money.
- Currency Conversion: Enable users to convert between multiple currencies within the wallet.
- Transaction History: Record details of user transactions, generate reports.
- Savings and Credit: Provide access to financial services like daily savings, individual/group credit, and group tontine.
- Biometric Authentication: Secure user accounts with biometric login options.
- Report generation tool: Let the LLM use account history to detect any anomolies, security risks or recommendations for different account or investment products. This acts as a helpful AI assistant for the user. Note: the suggestions it gives is for informational purposes only and doesn't act on the users behalf.

## Style Guidelines:

- Primary color: Vert Confiance (#0E5A59) for backgrounds of headers, active icons, and main buttons. This green hue symbolizes trust and stability, which is essential for a financial application.
- Accent color: Orange (#FF8C00) for 'Start' buttons and bonus/accent elements. This vibrant orange is used sparingly to draw attention to key actions.
- Background color: Light Gray (#F5F5F5) to maintain a clean and uncluttered interface.
- Titles and CTAs: 'Montserrat' (sans-serif) Bold for a modern and impactful look.
- Body text and labels: 'Roboto' (sans-serif) Regular for clear readability in financial contexts.
- Use bi-color (Vert Confiance and Orange) icons with a minimalist, rounded style. Use icons to represent main wallet functionality.
- Apply fluid transitions and fade-in animations for UI elements to enhance the user experience.
