================================================================================
                              OVERVIEW
================================================================================

eNkamba AI is an intelligent assistant module integrated into the eNkamba 
platform. It allows users to ask questions and get formatted responses with 
support for web search, deep analysis, and response export.

KEY FEATURES:
- Real-time conversational chat
- Response streaming
- Integrated web search
- Export to Word/PDF/Excel
- Conversation history
- Firestore storage
- Responsive interface

================================================================================
                         COMMON ISSUES
================================================================================

Issue: "GROQ_API_KEY not found"
Solution: Add GROQ_API_KEY to .env.local

Issue: "Firebase not initialized"
Solution: Verify all Firebase variables in .env.local

Issue: "Model decommissioned"
Solution: Use gemma2-9b-it model

Issue: "Firestore permission denied"
Solution: Check Firestore security rules

Issue: Slow performance
Solution: Add Firestore indexes, use pagination
